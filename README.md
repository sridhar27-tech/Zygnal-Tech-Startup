## ✨ Features

- ⚡ **Modern Angular 17** architecture with standalone components
- 🎨 **Tailwind CSS** + custom design system with dark mode support
- 🌌 **Three.js** powered 3D scenes and interactive visuals
- ✨ **GSAP** professional-grade scroll animations and timelines
- 📧 **EmailJS** integration for contact forms
- 📱 Fully responsive & mobile-first design
- ⚙️ Optimized production build with strict TypeScript
- 🔄 Hot reload development experience

## 🚀 Quick Start

### Prerequisites

- Node.js (v18+)
- Angular CLI (`npm install -g @angular/cli`)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/zygnal-tech.git
cd zygnal-tech

# Install dependencies
npm install

# Start development server
npm start
