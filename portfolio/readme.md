## About Us

Zygnal is an **AI-native platform** that unifies customer support, knowledge management, and engineering workflows. We close the loop between customers, support teams, and engineers by connecting tickets, code (GitHub), documentation, and AI insights in one place.

- **Living Help Docs** that stay fresh automatically
- **Learned Support Memory** from every resolved ticket
- **Deep GitHub Integration** — branches, PRs, and code context inside tickets
- **In-product AI Drawer** with contextual help and Vision support
- **MCP Server** for IDEs (Cursor, Claude Code, Windsurf)

Built in Sydney, hosted in AWS ap-southeast-2 with strong data residency and security focus.
